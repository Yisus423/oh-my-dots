# 🎨 Estética "Ayu Gold"

Paleta centralizada para i3, Polybar, Alacritty y Neovim.

| Elemento | Hexadecimal | Uso principal |
| :--- | :--- | :--- |
| **Background** | `#0a0e14` | Fondo de terminal, i3, Polybar y Neovim |
| **Background Alt** | `#1f2430` | Módulos activos, transiciones "Mirage" |
| **Foreground** | `#b3b1ad` | Texto principal, iconos inactivos |
| **Primary (Gold)** | `#e6b450` | Acentos, bordes de ventanas activas, advertencias, batería llena |
| **Secondary (Cyan)**| `#59c2ff` | Animaciones de carga, indicadores de teclado |
| **Alert (Red)** | `#ff3333` | Errores, batería crítica (`low-at = 15`) |
| **Disabled** | `#475266` | Texto inactivo, separadores de Polybar (`|`) |

> **Nota tipográfica:** El sistema utiliza `JetBrainsMono Nerd Font` (Medium, Size 13) para garantizar la correcta renderización de todos los glifos de Polybar y la terminal.
